# Sprint Insight – MTN FBB Sprint Review (Vite + React + TS)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Vercel
1. Push this folder to GitHub
2. In Vercel: New Project → Import repo
3. Framework: **Vite** (auto)
4. Build command: `npm run build` (auto)
5. Output dir: `dist` (auto)
6. (Optional) Add `GEMINI_API_KEY` in Project → Settings → Environment Variables
7. Deploy
```
